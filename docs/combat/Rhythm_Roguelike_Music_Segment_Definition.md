# 🎼 多段式乐曲中的「段」是什么？

## 一、什么是“段”（Segment）

在本游戏中：

> **段（Segment） ≈ 现实音乐中的“结构段落”**

它不是一小节，也不是一个音符，而是：
- 一段具有明确功能的音乐结构

---

## 二、常见的音乐段落（对应关系）

| 游戏中的段 | 现实音乐术语 | 含义 |
|---|---|---|
| 起始段 | Intro | 建立节奏与氛围 |
| 铺垫段 | Build-up | 累积能量与期待 |
| 爆发段 | Drop / Chorus | 主要能量释放 |
| 变化段 | Bridge / Break | 调整或对比 |
| 收束段 | Outro / Resolution | 结束并稳定情绪 |

一首完整曲子通常包含 **4–6 个段落**。

---

## 三、为什么段是战斗单位？

根据统一设计理念：**战斗不是对抗敌人，而是在多种风格同时存在的世界中，通过稳定、清晰、持续的表达，让自己的声音成为主导。**

- 每个段：
  - 提供一次主要"表达窗口"（输出窗口）
  - 提供一次处理干扰的机会（其他流派的演奏行为）
- 玩家通过：
  - 更高的结构效率（表达更清晰）
  - 更少的段数（表达更稳定）
完成战斗（成为合奏中的主导声音）

---

## 四、段数与音乐质量的关系

- 段数少：
  - 结构紧凑
  - 听感完整
- 段数多：
  - 重复感增强
  - 音乐拖沓
